Postgres statistics.

## Features

* Dependency-free.
* Supportes versions (see https://www.postgresql.org/support/versioning/)
    - 9.4
    - 9.5
    - 9.6
    - 10
    - 11
    - 12

## Install

Go version 1.13

```
go get github.com/cristalhq/pgstats
```
